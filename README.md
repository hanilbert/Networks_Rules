# Networks_Rules

这个仓库用于存储个人使用的网络代理规则配置，包括 Clash、Surge 等工具的分流规则。

## 项目说明

本项目主要包含：

- Clash 规则配置
- Surge 规则配置
- 自定义分流规则
- 规则更新脚本（如果有）

## 目录结构

```
.
├── clash/          # Clash 相关配置
├── surge/          # Surge 相关配置
└── rules/          # 通用规则文件
```

## 使用说明

1. 规则文件可以直接导入到相应的客户端使用
2. 建议定期更新规则以保持最佳使用体验
3. 部分规则可能需要根据个人需求进行自定义调整

## 注意事项

- 本仓库仅用于个人使用
- 请遵守相关法律法规
- 建议定期备份您的规则配置
